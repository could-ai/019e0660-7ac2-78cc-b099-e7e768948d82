# TikTok Clone App

A Flutter application replicating the core UI of TikTok, featuring a vertical scrolling video feed, full-screen video overlay with interactions (likes, comments, share), and a custom bottom navigation bar.

## Features
- **Vertical Video Feed**: Infinite scrolling full-screen vertical feed using `PageView`.
- **Video Overlay UI**: Animated like buttons, comment counts, profile pictures, and floating action panels.
- **Main Navigation**: Custom bottom bar with animated styling for the add button.
- **Dark Theme**: Standard immersive dark theme for video consumption.

## Tech Stack
- Flutter
- Dart
- Navigation: Standard Flutter Routing
- Theming: ThemeData (Dark mode by default)

## How to Run
1. Ensure you have Flutter installed.
2. Run `flutter pub get` to fetch dependencies.
3. Run `flutter run` to launch the app on your preferred device or emulator.
